# Harcama Takip 

Python ile yazılmış basit bir kişisel harcama takip uygulaması. Terminal üzerinden çalışır, harcamalarınızı kategori bazlı kaydedip toplam harcamanızı görebilirsiniz.

## Özellikler

- ✅ Harcama ekleme (kategori + tutar)
- ✅ Tüm harcamaları listeleme
- ✅ Toplam harcamayı hesaplama
- ✅ Verileri JSON dosyasında kalıcı olarak saklama (program kapansa bile veriler kaybolmaz)
- ✅ Basit menü sistemi ile kullanım

## Nasıl Çalıştırılır

```bash
python harcama_takip.py
```

Program açıldığında karşınıza bir menü çıkar:
1.Harcama Ekle
2.Harcamaları Göster
3.Toplamı Göster
4.Çıkış

İlgili numarayı girerek işlem yapabilirsiniz.

## Kullanılan Teknolojiler

- Python 3
- `json` modülü (veri kaydetme/okuma)
